External API Reference

Integrate your applications with the Shri Machail Mata Yatra ecosystem. Our API provides real-time access to pilgrim booking data and registration verification.

Authentication

Pass your unique security key in the x-api-key header for all requests.

Example Header
x-api-key: 77bf06e49bcf1eb4874a15e0af8a9fd6ebb10aa3720bfa1b

Base URLs

Use the following endpoints based on your development stage.

Productionhttps://shrimachailmatayatra.com/api/v1/external
Local Devhttp://localhost:5001/api/v1/external

Search Bookings

Retrieve all bookings made by a specific pilgrim via mobile number.

Request Details

GET /search
ParameterTypeDescription
mobilereq String 10-digit mobile number.
Response Example
{
  "success": true,
  "user": { "name": "...", "mobile": "..." },
  "bookings": [
    {
      "referenceId": "MATA/2026/0123",
      "darshanDate": "2026-05-20",
      "totalMembers": 4
    }
  ]
}

Get Booking Details

Fetch comprehensive details including all group members for a specific booking.

Request Details

GET /booking/:referenceId
ID VariableTypeDescription
referenceId String The unique Reference ID.
Response Example
{
  "success": true,
  "data": {
    "referenceId": "MATA/2026/0123",
    "members": [
        { "name": "Aditya", "regNo": "..." }
    ]
  }
}

Errors

CodeDescription
401Unauthorized. API key is missing or invalid.
403Forbidden. Scope permissions issue.
404Not Found. Data does not exist.
